# McUCA 

Aplicación web en Java para la gestión de un establecimiento de comida rápida.

### Herramientas-Tecnologías utilizadas

- Java SE Development Kit 8
- Vaadin
- Spring Boot
- MySQL Community Server 5
- Apache Maven 3
- Apache Tomcat 8
- Selenium
- JUnit
- Git
